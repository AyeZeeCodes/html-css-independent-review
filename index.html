Animal shelter project

For the animal shelter project we used HTML & CSS to practice making a column layout with floats and percentages. This project was basic but showed us why doing your own column layout can be challenging, and using a CSS framework like Boostrap can save a lot of time.

Bootstrap Wiki Project

This project's goal was to replicate the Wikiepedia home page using Boostrap classes. The project's main objective was to get us comfortable with Bootstrap's grid layout and some of the built in CSS compontents. It also taught us how to override Bootstrap styles with our own CSS, which is being used in this project.

Band Page Project

This project used HTML & CSS and Git to create one of the most common layouts on the web. We have a sidebar with some tour information and a main column with band images. We used floats and percentage based columns to acheive this. The goal of this project was to use floats and other CSS properties.
